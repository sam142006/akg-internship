# GlampQuest

GlampQuest is a single-page web application for discovering and booking unique outdoor stays like cabins, treehouses, domes, and houseboats.

## What it includes
- Live API integration with `https://demohotelsapi.pythonanywhere.com/hotels/`
- Explorer tab with search, location filter, price range, rating, sort, and pagination
- Bookings panel showing active reservations
- Favorites tab to save preferred stays
- Admin panel to add local cabin overrides
- Custom detail modal, toast alerts, and confirmation popups

## Development
Install dependencies:

```bash
npm install
```

Start the app locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Notes
The app is built with vanilla JavaScript, Bootstrap 5, and clean CSS overrides. It does not use React.
